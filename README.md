#### 介绍

Task是**通用功能**封装及管理的库。可以直接实现Task接口并应用;也可以用Control串联执行多个task，
并在执行过程中增删改查task。使用场景：按钮防双击、登录或认证状态、sdk外部功能的实现、流程控制等等。
